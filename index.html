How to Perform Basic Data Analysis in Python

A beginner's guide to analyzing data with Python

Introduction

Why learn this? Python is one of the most widely used languages for data analysis, and learning the basics helps users complete academic projects, explore personal data, and begin developing data science skills.

What You'll Learn

In this guide, you will learn how to install Python libraries, load a dataset, inspect and clean the data, run simple analysis, create visualizations, and save the results.

Who This Is For

üë§ Target Audience

Beginners who want to learn how to perform simple data analysis using Python for schoolwork, personal projects, or introductory data science practice.

üìö Prerequisites

You should know how to install basic software and open files, but don't need to know about Python libraries, data importing, or dataset cleaning.

üéØ Expertise Level

Beginner-level Python users with minimal familiarity with data analysis.

üè† Setting

These instructions are designed for calm, non-emergency environments such as at home, in a computer lab, or in a classroom.

Materials & Equipment Needed

  • A computer with Python 3.8 or newer installed
  • Internet access (recommended for installing libraries)
  • Jupyter Notebook, Google Colab, or a Python IDE (VS Code, PyCharm, etc.)
  • A CSV dataset saved on the computer
  • Python libraries: pandas, numpy, matplotlib

‚ö†Ô∏è Important Information

  • Installing Python or libraries may require administrator permissions.
  • The dataset should not include sensitive or personal information.
  • File names must be spelled correctly to avoid "FileNotFoundError" issues.
  • Save work frequently to avoid losing progress.

Step-by-Step Instructions

1

Install the Required Python Libraries

Open a terminal or command prompt and install the needed libraries:

pip install pandas numpy matplotlib
2

Open a Python Notebook or Script

Launch Jupyter Notebook, Google Colab, or a Python IDE. (For beginners who do not have an IDE installed, VS Code is a free option.)

3

Import the Data Analysis Libraries

At the top of your file, enter:

import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
4

Load Your CSV Dataset

Place your CSV file in an accessible location and load it:

df = pd.read_csv("dataset.csv")
5

Inspect the Dataset

Preview the data using:

df.head()
df.info()
df.describe()
6

Clean Missing or Duplicate Data

Remove incomplete or repeated entries:

df = df.dropna()
df = df.drop_duplicates()
7

Perform Basic Data Analysis

Run basic calculations such as averages or correlations:

df["column_name"].mean()
df.corr()
np.mean(df["column_name"])
8

Visualize the Data

Create a basic plot:

plt.plot(df["column_name"])
plt.show()
9

Save the Cleaned or Analyzed Dataset

Export your file:

df.to_csv("cleaned_data.csv", index=False)

üìà Data Analysis In Depth

1. What is Data Analysis?

Data analysis is the process of examining, transforming, and modeling data to discover useful information, draw conclusions, and support decision-making. It involves:

  • Data Collection: Gathering raw data from various sources
  • Data Cleaning: Removing errors, duplicates, and inconsistencies
  • Exploratory Analysis: Understanding patterns and distributions
  • Statistical Analysis: Computing metrics and relationships
  • Visualization: Creating charts to communicate findings
  • Interpretation: Drawing meaningful conclusions from results

2. Key Statistical Concepts

Mean (Average)

The sum of all values divided by the number of values. Represents the "typical" value.

Mean = (92 + 78 + 95 + 81 + 88) / 5 = 86.8

Use Case: Calculate average student grades or typical sales figures

Median

The middle value when data is sorted. Less affected by outliers than mean.

Sorted: 78, 81, 88, 92, 95 ‚Üí Median = 88

Use Case: Better for data with extreme values (e.g., household income)

Standard Deviation

Measures how spread out data is from the average. Higher values mean more variation.

Low std dev: Data is clustered close to mean

High std dev: Data is scattered far from mean

Use Case: Understand consistency in grades, test scores, or product quality

Correlation

Measures the relationship between two variables (-1.0 to 1.0):

  • +1.0: Perfect positive correlation (both increase together)
  • 0.0: No correlation (independent variables)
  • -1.0: Perfect negative correlation (one increases, other decreases)

Example: Math and Science scores often correlate positively (0.85)

Important: Correlation ‚â† Causation. Just because variables correlate doesn't mean one causes the other.

Percentiles & Quartiles

Percentile: Value below which a percentage of data falls

Quartiles: Divide data into 4 equal parts

  • 25th percentile (Q1): Lower quartile
  • 50th percentile (Q2): Median
  • 75th percentile (Q3): Upper quartile

Use Case: Understand distribution and identify outliers

3. Data Cleaning Best Practices

Handle Missing Values

# Check for missing values
print(df.isnull().sum())

# Option 1: Remove rows with missing values
df = df.dropna()

# Option 2: Fill missing values with mean
df['column'] = df['column'].fillna(df['column'].mean())

# Option 3: Fill with specific value
df['column'] = df['column'].fillna(0)

Handle Duplicates

# Find duplicate rows
duplicates = df.duplicated()
print(f"Duplicate rows: {duplicates.sum()}")

# Remove duplicates
df = df.drop_duplicates()

# Remove duplicates based on specific column
df = df.drop_duplicates(subset=['Student'])

Detect and Handle Outliers

# Method 1: IQR (Interquartile Range)
Q1 = df['column'].quantile(0.25)
Q3 = df['column'].quantile(0.75)
IQR = Q3 - Q1
outliers = df[(df['column'] < Q1 - 1.5*IQR) | (df['column'] > Q3 + 1.5*IQR)]
print(f"Outliers found: {len(outliers)}")

# Method 2: Z-score (values > 3 std devs from mean)
from scipy import stats
z_scores = np.abs(stats.zscore(df['column']))
outliers = df[z_scores > 3]

4. Data Visualization Guide

Bar Chart

Purpose: Compare values across categories

Best for: Comparing average sales by region, student grades by subject

Histogram

Purpose: Show distribution of continuous data

Best for: Understanding data spread, finding patterns

Scatter Plot

Purpose: Show relationship between two variables

Best for: Identifying correlations and clusters

Line Graph

Purpose: Show trends over time

Best for: Stock prices, temperature changes, website traffic

Box Plot

Purpose: Show distribution, median, and outliers

Best for: Comparing distributions across groups

5. Common Data Analysis Workflows

Finding Trends

# Time-series trend analysis
df_sorted = df.sort_values('Date')
plt.plot(df_sorted['Date'], df_sorted['Value'])
plt.title('Trend Over Time')
plt.show()

# Year-over-year comparison
yearly_avg = df.groupby(df['Date'].dt.year)['Sales'].sum()
print(yearly_avg)

Comparing Groups

# Group by category and calculate statistics
grouped = df.groupby('Category').agg({
    'Value': ['mean', 'sum', 'std'],
    'Count': 'sum'
})
print(grouped)

# Visualize group comparison
df.boxplot(column='Score', by='Team')
plt.show()

Identifying Patterns

# Correlation analysis
correlation_matrix = df.corr()
print(correlation_matrix)

# Find strongest correlations
for col1 in df.columns:
    for col2 in df.columns:
        if col1 != col2:
            corr = df[col1].corr(df[col2])
            if abs(corr) > 0.7:
                print(f"{col1} & {col2}: {corr:.2f}")

Troubleshooting & FAQ

Cause: Python was not added to PATH.

Fix: Reinstall Python and select "Add Python to PATH," then restart the terminal.

Problem 2: "FileNotFoundError" when loading the CSV

Cause: File name or path does not match.

Fix: Check the spelling, confirm the folder location, or use a full file path.

Problem 3: "ModuleNotFoundError: No module named 'pandas'"

Cause: Required libraries were not installed.

Fix: Install the libraries again:

pip install pandas numpy matplotlib

Problem 4: Plot does not appear

Cause: Some environments require an explicit display command.

Fix: Make sure to include:

plt.show()

Problem 5: Syntax or Indentation Errors

Cause: Missing quotes, parentheses, or incorrect spacing.

Fix: Check for typos or inconsistent indentation.
